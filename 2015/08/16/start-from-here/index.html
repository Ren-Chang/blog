Start From Here

最近总在想着，什么时候忙完老板的事情做一下自己的事，改进铁路地图，搭个博客，回家学车等等。就在昨天组内每周体育时间过后的疲惫的晚上，列出了这样的计划：

  • 搭建博客
  • 部署并测试铁路网PostgreSQL数据库
  • 编写铁路网查询API 改善铁路地图注记
  • 编写铁路网编辑API
  • 改写OSM的在线编辑器和瓦片渲染

于是乎今天就开始第一步。其实完全不知道为什么要搭建博客，因为总感觉自己写的东西并不是很有必要公开。大概只是为了后边铁路图动态页面的服务器和备案打个基础？至于为什么备案，大概是因为能搭载PostGIS的PostgreSQL主机并不多吧。

也许这些在今后看来都是孤陋寡闻导致的笑话，但开始总是好的。像高中宿舍日记的开头一样，

let’s start from here.
